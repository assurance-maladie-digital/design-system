### Exemples

#### Afficher le type d'un fichier

Vous pouvez afficher le type d'un fichier en utilisant la propriété `type`.
Les types supportés sont `pdf`, `word`, `excel`, `powerpoint`, `image`, `video`, `audio`, `archive`, `text` et `other`.
