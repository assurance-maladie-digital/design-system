# vite-vue-lib-template

> Fully automated.

-   [x] Eslint
-   [x] Prettier
-   [x] Conventional Commit
-   [x] Pre-commit
-   [x] Lint-staged
-   [x] Semantic-release
-   [x] TypeScript

## How to use:

1.  Clone the project
2.  `yarn`
3.  `npx semantic-release-cli setup`
4.  Add components
5.  `git add .`
6.  `yarn commit`
7.  `git push`
